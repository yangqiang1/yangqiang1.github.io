hexo解决插入图片问题

  • hexo完美插入图片

很多人在利用hexo搭建自己的个人博客，因为它非常简洁实用（据说是一个台湾在校生搞出来的，不说了，都是泪）。前不久我自己也搭建了一个，发布博文一切正常，但是在博文中插入图片时遇到了一些问题。很奇葩就是首页能正常显示图片，但是点开正文显示不了。最后经过查阅资料，发现一种方法，能够完美解决hexo本地插入图片的问题。好了，废话不多说直接来干货。
1、配置_config.yml（注意这个文件不是主题里面的），将post_asset_folder参数改为true，如下：

2、安装插件，打开Git bash,输入
npm install https://github.com/CodeFalling/hexo-asset-image – save
插件安装成功后，新建一个博文，在Git bash中输入“hexo new 插入图片”,这样的新建的好处是hexo会自动生成一个文件夹存放图片，如下图：

3、引入图片，使用下列标签

我们测试一下，放一张大雄的图片

4、查看，在Git bash中输入hexo g、hexo d，由此插入图片就完美解决了。
