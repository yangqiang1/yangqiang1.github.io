关于笔试，我有几点要总结一下：
1.块级元素：
<1>长度和高度都是可控的
<2>在没有设定其宽度的情况下，默认为100%
<3>总是从第一行开始
<4>块级元素中可包括行内元素和块级元素。
常见的有：p标签、h标签、ul无序标签、ol有序标签、form表单、div、hr水平分界线等等，
行内元素和块级元素正好相反。常见的有：br换行标签、a标签、img标签、input标签、select标签等等
块级元素和行内元素之间可以相互转换，行内->块级：display：block；块级->行内：display：inline，但是这并不代表你可以这样让两个div元素并排显示。
2.消除浮动：通常我们为了布局的需求，都会用到浮动并且还要消除它带来影响，我查找了相关资料，理解一下：一般情况下块级元素都处于标准流中，加上浮动之后就会脱离标准流浮起来，来看图说话：

左浮动处于同一流中，所以并排显示，上面的元素处于标准流中，所以独占一行。有一点，如果浮动元素下面有一个块级元素，那么它就会顶上来，占据浮动元素的下面空间。

现在比如是我们需要黑色盒子做底部，现在需要用到第一种消除浮动即在div4和div2与3中间添加一个div，样式定义为clear：both。看图：


第二种浮动,简单说一下，父div只设置颜色属性，两个子div左浮动，就会出现父div撑不起来消失不见的情况，我们可以通过设置父div{overflow或者hidden}来实现子div撑起父div的情况。
3.div居中

<3.1>假定父div中一个子div，让子div水平并且垂直居于父div中间，代码如下：
.p{
width: 100%;
height: 500px;
border:5px solid red;
position: relative;
}
.s1{
width: 200px;
height: 100px;
background-color: rgb(0, 255, 170);
margin: auto;
position: absolute;
top: 0;left: 0;bottom: 0;right: 0;
}
